# **App Name**: Ogen Shaket

## Core Features:

- User Onboarding: Allows users to input their name and select a gender for personalized content presentation.
- Category Browsing: Displays a grid of distinct categories of psychological tools and practices.
- Interactive Practice Cards: Presents each psychological practice as a flip card, showing a concise title on the front and detailed information on the back.
- Detailed Practice Insights: Provides 'Why it works' (scientific rationale), 'Steps to perform', and 'Practical tip' sections for each practice on the card's reverse.
- Card Navigation: Enables users to effortlessly navigate between practice cards within a chosen category using intuitive 'Next' and 'Previous' buttons.
- Gender-Adapted Content: Dynamically adjusts the text phrasing on the practice cards based on the user's selected gender (male or female).
- Progress Tracking: Visualizes user's progress through the cards in a category via a clear progress bar and numeric indicator.

## Style Guidelines:

- Color Anchor: 'Reassurance and Calm,' embodying the app's mission to provide psychological support. Scheme: Light.
- Primary color: A confident yet soothing Indigo (#6366F1). This hue, already prominent in the existing code, signals reliability and serves as a strong brand identifier, contrasting beautifully with the light background for key interactive elements.
- Background color: A soft, off-white with a subtle blue-violet tint (#F7F8FC). Derived from the primary hue but heavily desaturated, it provides a clean, open, and serene canvas that reduces visual fatigue and enhances readability.
- Accent color: A gentle Sky Blue (#99DEF5). This color, analogous to the primary but lighter and slightly less saturated, offers a refreshing contrast for secondary actions, subtle highlights, and a touch of optimistic clarity.
- Headlines and card titles: 'Alegreya' (humanist serif) for an elegant, intellectual, and inviting tone, reflecting the thoughtful nature of the content.
- Body text: 'Inter' (grotesque-style sans-serif) to ensure high legibility and a modern, objective feel, suitable for the educational descriptions and steps of the practices.
- Maintain the current set of 'Lucide' icons as demonstrated in the provided code, leveraging their clean lines and thematic relevance to psychological states (e.g., AlertCircle, Wind, Shield, Brain, Heart, Moon).
- Card-centric layout with generous padding and clear visual hierarchy for content, ensuring that each psychological practice feels digestible and accessible.
- Smooth and deliberate transitions, such as the existing card flip animation, to create a sense of focused engagement and reinforce the feeling of unfolding discovery within each practice.
